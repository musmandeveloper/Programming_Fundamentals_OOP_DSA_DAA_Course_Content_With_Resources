C++ Programming Course Roadmap

Year 1 – Fundamentals of Programming

Module Topics Duration
M1 Introduction to Programming, C++ History, IDE Setup 2 Weeks
M2 Syntax, Variables, Data Types, Input/Output 2 Weeks
M3 Operators & Expressions (Arithmetic, Relational, Logical) 2 Weeks
M4 Control Flow: if, else, switch-case 2 Weeks
M5 Loops: for, while, do-while, Nested Loops 3 Weeks
M6 Functions: Definition, Scope, Return types 3 Weeks
M7 Iterative vs Recursive thinking 2 Weeks
M8 Stack Basics (using arrays & recursion call stack) 3 Weeks
M9 Arrays & Strings 3 Weeks
M10 Revision, Quizzes & Mini Projects 2 Weeks
M11 Git & GitHub Basics 2 Weeks

Year 2 – Object-Oriented Programming (OOP)

Module Topics Duration
M1 OOP Introduction, History, Procedural vs OOP 2 Weeks
M2 Classes & Objects 2 Weeks
M3 Constructors & Destructors 2 Weeks
M4 Encapsulation & Abstraction 2 Weeks
M5 Inheritance (Single, Multiple, Multilevel) 3 Weeks
M6 Polymorphism: Compile-time & Runtime 3 Weeks
M7 Function and Operator Overloading, Friend Functions 3 Weeks
M8 Pointers with Objects 2 Weeks
M9 File Handling (Read/Write) 3 Weeks
M10 Revision, Mini Projects & GitHub Updates 2 Weeks

Year 3 – Data Structures & Algorithms (DSA)

Module Topics Duration
M1 Introduction to DSA, Features, History 1 Week
M2 Time & Space Complexity (Big O) 2 Weeks
M3 Arrays (1D & 2D), Searching & Sorting 3 Weeks
M4 Stack (Implementation & Applications) 2 Weeks
M5 Queue (Linear, Circular, Priority) 2 Weeks
M6 Linked List (Singly, Doubly, Circular) 3 Weeks
M7 Recursion in DSA & Backtracking Intro 2 Weeks
M8 Trees: Binary Trees & BSTs 3 Weeks
M9 Graphs: DFS, BFS, Representation 3 Weeks
M10 Advanced Sorting Algorithms (Quick, Merge) 3 Weeks
M11 Revision, LeetCode Challenges, Final Projects & GitHub Updates 2 Weeks

Additional Details

  • Exercises & Challenges: Each module includes 40-50 practice problems, LeetCode & HackerRank challenges, and small debugging exercises.
  • Mini Projects: Real-world projects such as a Grade Calculator, Library System, Bank Management, and Maze Solver to build a strong portfolio.
  • GitHub Portfolio: Regular updates with individual repositories, detailed READMEs, and project documentation.
